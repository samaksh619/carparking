# Car Parking

Simple C++ program stored in `park.cpp` (single-file example).

## Description

This repository contains a small C++ program for demonstrating car parking logic. The main source file is `park.cpp`.

## Build & run (Windows / PowerShell)

Using MinGW / g++:

```powershell
# compile
g++ -std=c++17 -O2 -Wall park.cpp -o park
# run
.\park
```

Using Microsoft Visual C++ (Developer Command Prompt):

```powershell
cl /EHsc park.cpp
park.exe
```

If you prefer to build with an IDE (Visual Studio, CLion), open the folder or create a project and add `park.cpp`.
